As the name suggests, this is a terminal-based number guessing game that takes in user inputs and sees if it matches the number to be guessed. 

I've decided to add hints to reduce the element of luck and make the game more strategy-based instead. The game is also replayable and foolproof, meaning any user input is tested before it is used by the main program. This ensures the program does not crash when an "invalid" input is given.

I've primarily taken up this project to learn how modular programming works; how I can create multiple reusable functions and assemble them into a functional terminal game. Also, this program has taught me how to build real, functioning programs - as simple as this one is. To say that working on this project has helped me understand modular programming better is an understatement.
